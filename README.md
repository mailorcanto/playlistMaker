# PlaylistMaker

PlaylistMaker is a simple application developed as part of a system development course at SENAC in Brazil. The application uses the Last.fm API to search for song names and saves the results in a plain text file (.txt) for demonstration purposes. This project is intended for learning purposes only.

## Features

- Search for songs using the Last.fm API
- Save song search results to a text file
- Basic CRUD operations for user management (example feature for learning)

## Technologies Used

- Node.js
- Express.js
- MySQL
- Axios (for API requests)
- dotenv (for environment variables)
